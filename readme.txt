Semtoo is an automated text creation tool (in German only!). Semtoo ist eine Software zur automatischen Textrecherche und Texterstellung in deutscher Sprache.

== Description ==

Semtoo ist die Abkürzung für "Semantisches Tool". Semtoo ist eine Software, die Texte aus dem Internet und eigene Textdokumente zusammenfasst und umformuliert. Semtoo übernimmt 75 - 90% der Arbeit eines Textschreibers. Je nach Anforderungsprofil an die Qualität des Textes muss das Resultat aber noch nachbearbeitet werden. Die Software produziert (noch) keine fehlerfreien Texte.

Um Semtoo nutzen zu können, ist es notwendig sich auf der Internetseite www.semtoo.com zu regstrieren. Die Anmeldung und Texte bis 350 Wörtern sind kostenlos. Darüber hinaus gibt es kostenpflichtige Accounts.
Sie können einen Auftrag zur Texterstellung sowohl im Plugin als auch auf der Webseite www.semtoo.com einstellen. Die Zusammenfassung wird auf externen Rechnern (der Cloud) durchgeführt. Ihr Auftrag kommt in eine Warteschlange und wird Ihnen nach Fertigstellung zugemailt und ist danach im Administrationsbereich von Wordpress verfügbar.

= Features =

* Automatische, zeitgesteuerte Erstellung passender, thematischer Blogtexte
* Verschiedene Quellen: Internetseiten, eigene Dokumente, Datenbanken
* Recherche nach Suchbegriffen und thematische Analyse von Internetseiten
* Filterung von Spam- und Marketingtexten
* Extrahierung der wichtigsten und relevanten Sätze und Aussagen in Texten
* Reduktion auf die wichtigsten Aussagen anhand von statistischen Analysen
* Vermeidung doppelter Aussagen
* Umstellung von grammatikalischen Satzbestandteilen, Passiv und Aktiv


== Installation ==

1. Laden Sie das Plugin herunter und aktivieren es
2. Melden Sie sich auf der Seite www.semtoo.com an
3. Loggen Sie sich dort ein und geben Ihren API-Key hier ein
